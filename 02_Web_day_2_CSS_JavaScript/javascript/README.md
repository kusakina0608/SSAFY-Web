## Javascript Type

`typeof()`를 사용하여 자료형을 확인할 수 있다.

javascript의 대표적인 데이터 타입은 다음과 같다.

- Number 객체
- String 객체
- Boolean 객체
- Array 객체
- Function 객체



#### var 대신 사용하는 문법

`let` C언어 변수와 비슷.

`const` C언어 const와 비슷.

#### JavaScript에서 ES
